# Scikit-learn Practice Repository

This repository contains a collection of machine learning examples, exercises, and projects using `scikit-learn`. It covers various aspects of the ML pipeline including data preprocessing, model selection, classification, regression, and ensemble methods.

## Repository Contents

The project includes notebooks and scripts for:

*   **Basics & Preprocessing**:
    *   `1_sklearn_start.ipynb`: Introduction to scikit-learn.
    *   `4_sklearn_PreProcess.ipynb`: Data preprocessing techniques.
    *   `8_polynominal_Feature.ipynb`: Polynomial feature generation.
    *   `Kaggle데이터셋_다운로드_NEW.ipynb`: Dataset download util.

*   **Classification**:
    *   `3_SVM.ipynb`: Support Vector Machines.
    *   `5_sklearn_classification.ipynb`: General classification examples.
    *   `6_classification_Optuna.ipynb`: Classification tuning with Optuna.
    *   `Plus_1_sklearn_wine_classification.ipynb`: Wine dataset classification.

*   **Regression**:
    *   `9_LinearRegressionModel.ipynb`: Linear Regression models.
    *   `Plus_5_LinearRegression_polyNorm.ipynb`: Polynomial regression.

*   **Ensemble Methods**:
    *   `10_ensemble.ipynb`: Ensemble learning techniques.
    *   `Plus7_ensenble.ipynb`: Additional ensemble examples.
    *   `Plus_7_ensemble_gemini.ipynb`: AutoGluon and ensemble tuning.
    *   Voting outcomes (CSV files).

*   **Projects & Specific Datasets**:
    *   `7_Titanic.ipynb` & Titanic related notebooks: Titanic survival prediction.
    *   `Plus_3_sklearn_digits.ipynb`: Digits dataset analysis.
    *   `Plus_4_sklearn_red_wine_quality.ipynb`: Red wine quality analysis.

*   **Unsupervised Learning**:
    *   `13_unsupervisedLearning.ipynb`: Unsupervised learning examples.

## Environment & Usage

**Important Note**: This repository is designed to be used with **Antigravity**. 

Please run the code and notebooks using the Antigravity environment instead of standard Jupyter Notebooks to ensure compatibility and access to necessary tools.
